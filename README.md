# Netflix_titles movies and tv shows Data Analysis using SQL

![Netflix logo] (https://github.com/narurock91-cmd/Netflix_titles-sql-project/blob/main/download%20(1).jpeg)

## Objective : This project focuses on analyzing a Netflix dataset that includes detailed information about various titles such as movies, TV shows, dramas, and web series. The dataset contains attributes like title type, director, cast, country of production, and rating.The analysis gives  insights that can help understand viewing preferences, content diversity, and platform trends.
